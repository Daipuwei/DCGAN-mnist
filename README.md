# DCGAN-mnist
利用简单DCGAN生成手写数字图像
# 运行环境
 1. python3.6
 2.  tensorflow1.14 
 3. keras2.2.4 
 4. opencv3.4.5.20

# 运行流程
在命令行运行如下命令：

```
python main.py
```
